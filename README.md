Fast Secure Spend
================

###	Coin Spec
	* Algorithm : x11 (POS/MN)
	* Coin Ticker : FSS
	* Max supply : 30,000,000
	* Block : 30 sec
	* Staking-time : 2hrs
	* MN collateral : 3,000 FSS (40% POS / 60% MN)
	* Premine : 300,000 FSS
